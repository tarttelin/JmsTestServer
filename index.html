JMS Test Server - Integration testing for JMS

JMS Test Server is an in-VM JMS server that can be watched and asserted on by your integration tests.

Eliminates the need for sleeps or quirky monitoring of the application under test by allowing you to block on messages in the JMS queue until they have been processed.

Capture the messages added to the queue so you can assert on their content.

Block the queues so no messages are delivered, then choose the order in which to deliver the messages.

Example

As the library was written Test First, the repository is full of useful tests which serve as the most comprehensive documentation.

Dependency

The library is not yet available from Maven Central, but you can install it in your local maven cache by running gradle build install, then using the following dependencies in your project:

  • groupId: com.pyruby
  • artifactId: java-stub-server
